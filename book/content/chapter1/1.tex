\item 语言标准：基于现代 C++20 标准编写
\item 内存检查：推荐使用 valgrind（\url{https://valgrind.org/}）进行内存泄漏检测
\end{itemize}

为获得最佳实践体验，建议读者：
\begin{itemize}
\item 确保开发环境支持 C++20 标准
\item 安装最新版 valgrind 工具
\item 熟悉基本的 make 使用方式
\item 运行测试前先执行内存检查
\end{itemize}
